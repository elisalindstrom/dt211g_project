# DT211G Projektuppgift
NextTBR är en webbapplikation som visar kommande och publicerade böcker från författare.
Användare kan söka efter sina favoritförfattare och få upp en författarprofil, eventuella kommande boksläpp samt ett urval av redan publicerade böcker.
Det går även att spara böcker i en "Want to Read"-lista.

## Två API:er används:
Open Library API för att hämta författare
Google Books API för att hämta böcker

https://nexttbr.onrender.com/
